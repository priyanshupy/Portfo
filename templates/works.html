001 / AI Project

AI-Powered Code Repair Assistant

MSc Dissertation — 2025

Designed and developed an AI-powered automated code repair system using Python, LangChain, ChromaDB, and Retrieval-Augmented Generation (RAG) to analyse, repair, and explain Data Structures and Algorithms submissions.

  • Implemented semantic retrieval with embeddings and contextual prompting to generate minimal, explainable code fixes.
  • Engineered a multi-stage pipeline with AST-based validation — achieving 80%+ identifier retention vs a 50%+ baseline.
  • Awarded Distinction as part of the MSc; currently co-authoring a research paper on automated AI-assisted code repair.

Stack: Python, LangChain, ChromaDB, RAG, LLMs, AST analysis

002 / AI Project

AI Search Assistant (Perplexity-Clone)

2025

Built a Perplexity-inspired AI search assistant using Python, Streamlit, LangChain, ChromaDB, and Ollama to answer natural language questions using up-to-date information retrieved from the web.

  • Designed an end-to-end RAG pipeline: DuckDuckGo search → parse and chunk results → embed → index in ChromaDB → LLM inference.
  • Integrated a locally hosted Ollama language model to generate grounded responses, reducing hallucinations with fully local inference.
  • Containerised with Docker for reproducible deployment; collaborated with a small team on architecture and backend integration.

Stack: Python, Streamlit, LangChain, ChromaDB, Ollama, Docker
